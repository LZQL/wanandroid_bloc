# wanandroid_bloc

该项目我命名为wandroid_bloc，它是基于wanandroid.com网站的免费开放api编写，该网站收录了一些优秀的Android技术文章。可以方便的为开发者很好的学习。目前具有的功能点有，博文页，搜索页，项目页，项目分类页，公众号分类页，体系，文章详情页，分享，浏览器打开文章，个人中心，登录注册，我的收藏，切换语言，更换主题色，收藏，取消收藏功能。
项目名的bloc的意思是，通过bloc这个状态管理框架作为项目的状态管理部分，通过bloc进行页面UI和业务逻辑的代码分离，不仅对状态有一个非常良好的全局管理，并且代码结构也良好的去去耦合性。
